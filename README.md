# Haitian HDC II landing page demo

Landing page tĩnh cho dòng máy đúc áp lực buồng nguội Haitian HDC II.

## Nguồn nội dung

- Website mẫu: https://mayepphun.com.vn/
- Brochure PDF người dùng cung cấp: HDC Cold Chamber Die Casting Machine HDC200-HDC8800 II

## Nội dung đã dùng

- Dải sản phẩm HDC200-HDC8800 II từ bìa brochure.
- Bảng thông số chi tiết HDC200-HDC4500 II-SF.
- Ghi chú HDC5000-HDC8800 cần hỏi nhà sản xuất.
- Tính năng: áp lực hệ thống 19-21 MPa, hệ phun HP, điều khiển KEBA,
  smart mold opening/closing, kết cấu kẹp khuôn, giám sát chất lượng.
- Hotline/email lấy từ landing page mẫu.

## Thông tin cần bổ sung

- Xác nhận đơn vị phân phối HDC II tại Việt Nam.
- Xác nhận hotline/email/Zalo dùng cho máy đúc áp lực.
- Giá, lead time, tồn kho, điều kiện giao hàng và lắp đặt.
- Chính sách bảo hành, bảo trì, phụ tùng.
- Case study, ngành ứng dụng và ảnh/video thực tế tại Việt Nam.

## Chạy local

Mở `index.html` trực tiếp trong trình duyệt hoặc chạy một web server tĩnh trong thư mục này.
